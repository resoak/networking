# 網路範圍

## 由小到大

1. **PAN**: 個人範圍網路，例如手機連藍牙耳機。
2. **LAN**: 區域網路，例如家裡、教室、辦公室。
3. **MAN**: 都會網路，覆蓋一整個城市。
4. **WAN**: 廣域網路，跨城市甚至跨國家。

## 快速記法

- PAN = 個人
- LAN = 區域
- MAN = 城市
- WAN = 廣域

## 生活例子

- 耳機連手機: PAN
- 家裡 Wi-Fi: LAN
- 城市級網路建設: MAN
- 跨國連線: WAN

## 簡報補充

- 投影片有把四種網路範圍做更具體的例子：
  - `PAN`：個人隨身或數公尺範圍內的網路，例如手機、耳機、智慧手環。
  - `LAN`：同樓層、同棟建築或數棟建築物內，例如教室、辦公室、校園。
  - `MAN`：由數個區域網路組成的城市級網路，例如交通控制或市政網路。
  - `WAN`：跨縣市、跨國的網路，例如 `Internet`、`TANET`、`HiNet`、`SEEDNet`。
- 類型比較時可以再加上這個印象：
  - `PAN`：最短、最便宜
  - `LAN`：速度快、最常見
  - `MAN` / `WAN`：多半仰賴電信公司連線
