# 通訊模式

## 兩種常見模式

- **Client/Server**: 由中央伺服器提供服務，客戶端提出要求。
- **Peer-to-Peer**: 裝置彼此平等，可以直接互相分享。

## 快速判斷

- 有中心集中管理，多半是 Client/Server。
- 彼此直接共享，多半是 Peer-to-Peer。

## 比較

| 模式 | 特徵 | 例子 |
| --- | --- | --- |
| Client/Server | 有中心 | 網站、雲端服務 |
| Peer-to-Peer | 沒有固定中心 | 裝置直接分享 |

## 簡報補充

- 投影片把網路作業環境分成兩種常見架構：`Client/Server` 與 `Peer-to-Peer`。
- `Client/Server` 的概念是：
  - `Client` 負責使用資源
  - `Server` 負責提供資源
- 課堂例子是 `WWW`：
  - Client 端可以是 `IE`、`Chrome`、`Firefox`
  - Server 端可以是 `IIS`、`Apache`
- `Peer-to-Peer` 則強調節點地位相同，例如：
  - Windows 共享中心
  - 手機之間直接互傳資料

## 一句口訣

- `有中心提供服務 = Client/Server`
- `彼此平等互相分享 = Peer-to-Peer`
