# 常見技術

## 技術對照

- **Ethernet**: 有線區域網路標準，常見於網路線。
- **802.11**: 無線區域網路標準，也就是 Wi-Fi。
- **Bluetooth**: 短距離個人無線通訊。
- **DSL**: 利用電話線提供網際網路服務。

## 生活連結

- 網路線 = Ethernet
- 家用無線網路 = 802.11
- 耳機連手機 = Bluetooth
- 舊式電話線上網 = DSL

## 簡報補充

- 投影片把短距離常見技術分成兩大類：
  - 個人網路常見：`Bluetooth`
  - 區域網路常見：`Ethernet`、`IEEE 802.11`
- `Ethernet` 常見速率：`100Mbps`、`1Gbps`，更高還有 `10Gbps`、`100Gbps`。
- `IEEE 802.11` 就是大家熟悉的 `Wi-Fi`，有效傳輸距離大約可記成**約 100 公尺**。
- `Bluetooth` 可以先記成**近距離、低功耗**的個人裝置連線技術。

## 生活連結補充

- 手機接藍牙耳機：`Bluetooth`
- 教室或辦公室的有線網路：`Ethernet`
- 家中無線基地台提供的網路：`IEEE 802.11 / Wi‑Fi`
