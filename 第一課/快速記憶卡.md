# 快速記憶卡

## 範圍

- **PAN**: 個人範圍，想到手機連耳機。
- **LAN**: 區域範圍，想到家裡或學校網路。
- **MAN**: 都市範圍，想到整個城市。
- **WAN**: 廣域範圍，想到跨城市、跨國。

## 技術與模型

- **802.3**: 就是 Ethernet。
- **802.11**: 就是 Wi-Fi。
- **TCP/IP**: 網際網路最重要的通訊基礎。
- **OSI**: 標準參考模型，共 7 層。

## 補充快記

- Internet = 很多網路互連而成的全球網路。
- E-mail / Telnet / FTP = Internet 早期三大基本功能。
- Star = 架構簡單、維護容易。
- Bus = 共用主幹線。
- Client/Server = 有人提供資源、有人使用資源。
- Peer-to-Peer = 節點地位相同。
