# 核心名詞

## 重點

- **Network**: 多個裝置彼此連接，可以分享資料與資源。
- **Internet**: 全球很多網路連成的大型網路，主要使用 TCP/IP。
- **Server / Client**: Server 提供服務， Client 發出請求來使用服務。
- **bps**: bits per second，每秒可傳多少位元，是速度單位。

## 怎麼記

- 把 Internet 想成很多小網路互相連接。
- 先分清楚「網路」和「網際網路」不是同一個範圍。
- 看到服務提供者先想到 Server，看到使用者端先想到 Client。
