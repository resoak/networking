# 核心名詞

## 重點

- **Network**: 多個裝置彼此連接，可以分享資料與資源。
- **Internet**: 全球很多網路連成的大型網路，主要使用 TCP/IP。
- **Server / Client**: Server 提供服務， Client 發出請求來使用服務。
- **bps**: bits per second，每秒可傳多少位元，是速度單位。

## 怎麼記

- 把 Internet 想成很多小網路互相連接。
- 先分清楚「網路」和「網際網路」不是同一個範圍。
- 看到服務提供者先想到 Server，看到使用者端先想到 Client。

## 簡報補充

- 投影片把「網路」先當成**讓多台設備彼此連接、交換資料、共享資源**的基本架構。
- 課堂一開始特別強調：學網路不只是會上網，而是要慢慢理解**軟硬體架構、運作原理、協定、設定、設計與管理**。
- `Internet` 不只是很多網站，而是**由全球各地許多區域網路互相連接形成的超大型網路**。
- `TCP/IP` 是 Internet 的核心開放式協定架構：
  - `TCP` = Transmission Control Protocol
  - `IP` = Internet Protocol

## 一句話版本

- `Network`：把設備連起來。
- `Internet`：把很多網路再連起來。
- `TCP/IP`：讓 Internet 能運作的共同語言。
