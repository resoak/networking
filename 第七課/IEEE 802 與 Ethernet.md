# IEEE 802 與 Ethernet

## 重點

- `IEEE 802` 是一系列和區域網路相關的標準。
- 錄音中老師特別提到 `802.3` 與 `802.5`，表示早期 LAN 技術其實有競爭，不是只有一種做法。
- `802.3` 就是 Ethernet，現在是最常見的有線區域網路技術。
- `802.5` 是 Token Ring，屬於早期重要但後來被 Ethernet 壓過去的技術。
- 老師有提到早期 LAN 曾出現很多不同媒介與做法，但最後 Ethernet 因為成本、普及度與生態系勝出。

## 怎麼記

- 先把 `802.x` 想成「LAN 標準家族」。
- 看到 `802.3` 就直接想到 Ethernet。
- 看到 `802.5` 就想到它是歷史上有名、但現在比較少見的 Token Ring。

## 簡報補充

- 投影片補了 `IEEE 802` 和 `OSI` 的對應關係：
  - `802.1`：高階層介面
  - `802.2`：`LLC`
  - `802.3`：Ethernet
  - `802.11`：無線區域網路
  - `802.15`：個人無線網路，如 Bluetooth、Zigbee
- 所以 `802.3` 不只是「一種 LAN」，而是今天最重要的有線區域網路標準核心。
- 投影片也提到早期 LAN 標準曾有 `802.4`、`802.5` 等競爭者，但現今最常看到的是 `802.3`、`802.11`、`802.15`。
