# 常見網路威脅

## 為什麼要認識威脅？

防護策略要對應攻擊方式。先知道攻擊者可能竊聽、偽冒、竄改或中斷服務，才知道該用防火牆、加密、認證、備份或 IDS。

## 常見威脅整理

| 威脅 | 說明 | 主要破壞 |
|---|---|---|
| 竊聽／網路監聽 | 攔截封包取得帳密或敏感資料 | 機密性 |
| 偽冒身分 | 冒充使用者、主機或網站 | 認證、完整性 |
| 中間人攻擊 | 攻擊者插在雙方通訊中竊聽或竄改 | 機密性、完整性 |
| 密碼攻擊 | 暴力破解、字典攻擊、撞庫 | 帳號安全 |
| DoS／DDoS | 讓服務無法正常回應 | 可用性 |
| 惡意程式 | 病毒、蠕蟲、木馬、勒索軟體、間諜軟體 | 多種目標 |
| 網路釣魚 | 用假網站、假信件騙帳密 | 身分與機密性 |
| DNS spoofing／pharming | 導到假網站 | 認證、機密性 |
| SQL injection | 對資料庫輸入惡意指令 | 完整性、機密性 |
| XSS | 在網頁注入腳本攻擊使用者 | 使用者資料 |
| 零日攻擊 | 利用尚未修補或未知漏洞 | 系統安全 |
| 供應鏈攻擊 | 從軟體更新、外包或供應商下手 | 系統信任 |

## 課堂補充

影片中提到許多攻擊名稱，有些不是本章考試核心，但可用來理解現代資安問題：例如 USB baiting、cryptojacking、IoT botnet、typosquatting、Stuxnet 等。

## 防護對應

- 竊聽 → [[加密與金鑰]]、HTTPS/TLS
- 偽冒 → [[認證與數位憑證]]、數位簽章
- 竄改 → [[雜湊函數與資料完整性]]、數位簽章
- 未授權連線 → [[防火牆]]
- 可疑行為 → [[入侵偵測系統 IDS]]
- 資料被破壞 → 備份與復原

## 怎麼記

攻擊大多可以放進 CIA：

- 偷看資料：破壞機密性。
- 改資料：破壞完整性。
- 讓服務不能用：破壞可用性。
