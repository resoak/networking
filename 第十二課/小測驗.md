# 第十二課小測驗

## 一、選擇題

1. 下列何者不是資訊安全 CIA 三大目標？
   - A. 機密性
   - B. 完整性
   - C. 可用性
   - D. 監控性

2. 加密與解密使用同一把金鑰，稱為：
   - A. 公開金鑰密碼系統
   - B. 秘密金鑰密碼系統
   - C. 數位憑證
   - D. 入侵偵測系統

3. 雜湊函數最常用來檢查：
   - A. 資料完整性
   - B. 網路頻寬
   - C. IP 位址是否足夠
   - D. 路由成本

4. 下列哪一項最能提供不可否認性？
   - A. NAT
   - B. ARP
   - C. 數位簽章
   - D. DHCP

5. IDS 的主要功能是：
   - A. 分配 IP 位址
   - B. 偵測可疑入侵行為
   - C. 將私有 IP 轉成公有 IP
   - D. 計算路由表

## 二、填充題

1. 資訊安全三大目標為：__________、__________、__________。
2. 防火牆基本類型可分為 __________ 防火牆與 __________ 防火牆。
3. 公開金鑰密碼系統中，公開的是 __________，必須自己保管的是 __________。
4. 若有 n 個使用者使用秘密金鑰密碼系統互相加密通訊，整個系統需要 __________ 把共享金鑰。
5. 數位憑證通常由可信任的 __________ 簽發。

## 三、簡答題

1. 請用自己的話說明機密性、完整性、可用性的差別。
2. 為什麼防火牆不能取代加密？
3. 對稱式加密與公開金鑰加密各有什麼優缺點？
4. 數位簽章如何同時幫助確認身分與檢查資料完整性？
5. IDS 與防火牆在任務上有什麼差異？

## 參考答案

### 一、選擇題

1. D
2. B
3. A
4. C
5. B

### 二、填充題

1. 機密性、完整性、可用性
2. 封包過濾式、應用閘道式／代理式
3. 公鑰、私鑰
4. `n(n-1)/2`
5. CA／憑證授權中心

### 三、簡答題提示

1. 機密性是不讓未授權者看；完整性是不讓資料被改；可用性是需要時能使用。
2. 防火牆控管連線，加密保護內容；封包若被允許通過，內容仍可能需要加密保護。
3. 對稱式速度快但金鑰配送難；公開金鑰方便配送與簽章，但速度較慢。
4. 發送者用私鑰簽章，接收者用公鑰驗證；若內容被改，驗證會失敗。
5. 防火牆像門禁，負責擋或放行；IDS 像監視器，負責發現可疑行為並告警。
