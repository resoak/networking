# 第十二課目錄：網路安全

第十二課進入最後一章 **網路安全**。這一課從「為什麼需要安全」開始，整理資訊安全三大目標 [[資訊安全與 CIA|CIA]]，再往下說明常見威脅、防火牆、加密、金鑰、數位簽章、憑證與 [[入侵偵測系統 IDS|IDS]]。

> 本課筆記依據 `錄製 (23).m4a`、`錄製 (24).m4a` 與 [[第12章_網路安全_習題解答]] 整理。逐字稿有部分語音辨識誤差，專有名詞已依課本與課堂脈絡校正。

## 學習順序

1. 先讀 [[資訊安全與 CIA]]，建立「安全要保護什麼」的框架。
2. 再讀 [[常見網路威脅]]，理解攻擊者可能破壞哪些目標。
3. 接著讀 [[防火牆]]、[[加密與金鑰]]，掌握主要防護手段。
4. 最後讀 [[公開金鑰與數位簽章]]、[[雜湊函數與資料完整性]]、[[認證與數位憑證]]、[[入侵偵測系統 IDS]]，把身分、完整性、不可否認性串起來。
