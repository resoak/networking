# 網路傳輸設備

## 重點

- 設備可以依功能由簡到繁去記：
  `Modem → NIC → Repeater → Hub → Bridge → Switched Hub → Router → Layer 3 Switch → Gateway`

## 基本設備

### Modem

- `Modem` = `MOdulation / DEModulation`
- 功能是把數位與類比訊號互相轉換。

### NIC

- `NIC` = Network Interface Card
- 不同網路環境需要對應的網路卡。
- 現在很多主機板已內建。

## 延長與集中

### Repeater

- 用來把衰減的訊號整形、放大後再送出去。
- 重點是**延長傳輸距離**。

### Hub

- 可先把它想成**多埠中繼器**。
- 功能比單純 Repeater 多，但本質仍偏向訊號延伸。

## 區隔與交換

### Bridge

- 用來連接兩個網段。
- 讓網段之間比較不會彼此拖累。

### Switched Hub

- 又可以想成**多埠橋接器**。
- 每個埠都有獨立頻寬，效率比一般 Hub 好很多。

## 尋徑與轉換

### Router

- 核心能力是 `Routing`。
- 用來連接不同網路，幫資料找路。

### Layer 3 Switch

- 因為尋徑屬於 OSI 第三層，所以叫第三層交換器。
- 以硬體做部分尋徑，通常速度好、架構較單純。

### Gateway

- 具備更強的相容與轉換能力。
- 遇到不同架構或不同標準時，Gateway 的角色更明顯。

## 比喻記法

- `Hub` 像大家擠在同一個平面路口。
- `Switched Hub` 像立體交流道，可以同時分流。
- `Router` 像跨城市找路的導航。
