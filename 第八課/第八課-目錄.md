## 課堂錄音重點

- 這次錄音裡老師講得最細的部分，是 [[IP 位址與子網路]]。
- 特別反覆強調：
  - 十進位和二進位的轉換
  - IPv4 為什麼是 4 組 8 bits
  - `Network bits` 和 `Host bits` 到底在幹嘛
  - 怎麼用遮罩和 `AND` 判斷是不是同網段
  - 為什麼要切子網路、為什麼 `CIDR` 比死背 A/B/C 更實用
- 新增的兩段錄音補強了 [[DHCP 與 DNS]] 和 [[IPv4 標頭與 IPv6]]：
  - `DNS` 為什麼從早期 hosts 檔演變成分散式階層系統
  - `Root / TLD / Authoritative DNS` 的查詢流程
  - `DNS cache`、`A / AAAA / CNAME / MX / NS Record`、`DDNS`
  - `IPv4 header` 像信封，先交代來源、目的地、長度、協定與檢查資訊
  - `TTL`、`Protocol`、`Header Checksum`、分片與 `Wireshark` 觀察
