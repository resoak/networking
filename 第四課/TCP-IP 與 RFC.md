# TCP-IP 與 RFC

## TCP/IP 是什麼

- `TCP/IP` 是 Internet 世界實際最重要的協定族。
- 課堂提到：
  - `TCP`：Transmission Control Protocol
  - `IP`：Internet Protocol

## TCP/IP 的制定背景

- 早期由 `ARPA` 相關單位研究。
- `1983` 年，`TCP/IP` 正式納入 Internet 架構。
- `IAB` 之下的重要單位：
  - `IETF`：偏協定制定與中短期問題
  - `IRTF`：偏長期研究

## RFC 是什麼

- `RFC` = Request for Comments
- 是 Internet 技術文件的重要來源。
- 協定、演算法、機制說明，很多都能在 RFC 找到。

## 常見 RFC 例子

- `IP`：RFC 791
- `TCP`：RFC 793
- `UDP`：RFC 768
- `FTP`：RFC 959
- `DNS`：RFC 1034、1035

## 怎麼記

- `TCP/IP` 是 Internet 的實戰協定系統。
- `OSI` 比較像教學與分析架構。
- `RFC` 則像是 Internet 世界的重要技術文件庫。
